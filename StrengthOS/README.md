# StrengthOS - Landing Page

Landing page premium para StrengthOS, una plataforma de entrenamiento personalizado con acceso privado para clientes.

## 🎨 Diseño "Dark Luxury"
- **Color de Acento:** Gris (#545454)
- **Sector:** Salud / Fitness
- **Estilo:** Personal / App Landing

## 📁 Estructura
- `index.html`: Estructura semántica adaptada a Landing y Login.
- `styles.css`: Estilos con Glassmorphism y variables CSS dinámicas.
- `script.js`: Interacciones, Observer para scroll, Flip Cards en mobile, contador animado y sistema de partículas.
- `/images/`: Contiene todas las imágenes generadas por IA.

## 🖼️ Imágenes Incluidas
1. `logo.png`: Logo principal minimalist y moderno.
2. `hero_visual.png`: Render 3D de elementos fitness en alta calidad.
3. `service_1.png` - `service_6.png`: Imágenes fotorrealistas con iluminación premium gris para las Flip Cards.
4. `about_image.png`: Fotografía premium de entrenador.

## ✨ Animaciones y Features
- **Flip Cards:** Animación 3D en las tarjetas de características (toca en móvil, pasa el cursor en PC).
- **Glassmorphism:** Efecto de desenfoque aplicado a las tarjetas, formularios y navbar.
- **Partículas Aleatorias:** Pequeñas luces que flotan por la pantalla continuamente.
- **Scroll Observer:** Las secciones aparecen suavemente cuando haces scroll hacia abajo.

## 🚀 Uso
Simplemente abre `index.html` en cualquier navegador web moderno. No requiere empaquetadores ni dependencias externas.
